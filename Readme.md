# Validación automática de mediciones solares minuto a minuto

Este proyecto desarrolla una solución de Data Science y Big Data para validar automáticamente mediciones solares de H, GHI y DNI, clasificando si cada medición es correcta o incorrecta a partir de datos históricos procedentes de archivos Excel.

## Objetivo provisional

Desarrollar una arquitectura reproducible de ingesta, almacenamiento, procesamiento y modelado para analizar la calidad de datos solares minuto a minuto y detectar mediciones incorrectas.

## Datos

- Gráficas diarias 2023
- Gráficas diarias 2024
- Altura solar minuto a minuto en Sevilla

## Tecnologías previstas

- Python
- Pandas
- SQL / PostgreSQL
- Spark / PySpark
- Machine Learning
- Visualización
